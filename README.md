# Student Management System

This project is a Java-based application used to manage student records. It provides a simple GUI where users can add, update, delete, and search student details. The project uses MySQL database to store data permanently. It is built using Java Swing and JDBC.


# Student Management System

Java project using Swing GUI and MySQL database.

Features:
- Add Student
- View Student
- Update Student
- Delete Student
- Search Student

Run:
1. Run SQL file
2. Compile: javac *.java
3. Run: java StudentGUI
